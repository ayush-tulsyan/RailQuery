This demo application illustrates the use of AngularJS with Node Js, Neo4J Graph DB and Express. The application also uses JSON Web Tokens and Passport JS to protect the API endpoints. Use 'guest'/'guestpassword' to login in order to access the API calls. Currently PassportJS's LocalStrategy is implemented. Stay tuned for implementation of other strategies supported by PassportJs
Manage your list of drugs by navigating to the Manage Drugs section. You can add new drugs to the list by using the form. When a new drug is added, it will automatically be available in the drop downs for creating relationships
Once you have the drugs created in the graph database, you can then specify the relationships between them by using the relationship form. Select two drugs and the relationship - Interacts with or Alternative to - between them to create the relationship.
